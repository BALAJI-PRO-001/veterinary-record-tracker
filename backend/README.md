## Admin Login
**Api usage and request response details**
``` base
Request Information
===================
URL: /api/v1/admin/login
METHOD: POST
CONTENT-TYPE: application/json
BODY: {
    email: <string>
    password: <string>
}


Response (Request Based)
=========================
{
  success: true 
  statusCode: 200  
  message: "Admin Logged in successfully 
}
```
